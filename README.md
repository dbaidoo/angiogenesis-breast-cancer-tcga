# angiogenesis-breast-cancer-tcga
 Code and reproducibility materials for: Multi-Dimensional Structure and Subtype-Specific Heterogeneity of Angiogenesis in Breast Cancer

 # Multi-Dimensional Structure and Subtype-Specific Heterogeneity of Angiogenesis in Breast Cancer

This repository contains the R code used to reproduce the analyses reported in:

Baidoo D. et al.

## Data source

TCGA-BRCA RNA sequencing data obtained from:

https://portal.gdc.cancer.gov/

## Main analyses

- Expression preprocessing
- Hallmark angiogenesis gene set extraction
- Principal component analysis
- Kaplan–Meier survival analyses
- Cox proportional hazards models
- Proportional hazards diagnostics
- Time-dependent Cox models
- Nonlinearity assessment
- Tertile sensitivity analyses
- Multiple-testing correction
- E-value sensitivity analyses
- Subtype-specific analyses
- Gene Ontology enrichment analysis

## Software

R version 4.5.2

## Author

Dennis Baidoo
